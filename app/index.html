Heroku Apps Today

Written by MJ Grant on August 10, 2014.

Today I learned how to set up Heroku, tie it to a github project, and use it to stand up a simple blog app.

Some gotchas: don’t directly edit the .css, the build process will clobber it with scss contents.

Don’t edit dist or tmp directly. Edit the source - think of dist and tmp as compiled code. npm install && bower install <- chain like this when setting up a new project

Before we use heroku, we need a server. Express? Static node server? git init git add . git commit -m “generated a site with yo browserify” npm init or edit existing package.json npm install express --save (add to existing package.json) Create your server: vim server.js
